# Tibia Idle

## How to use

`git clone ...`</br>

Open Two Terminals

### In the first terminal

`cd server`</br>
`npm install`<br>
`npm run start`


### In the second terminal

`cd client`</br>
`npm install`<br>
`npm run serve`</br>
